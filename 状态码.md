- 状态码

    -  1xx 请求正在处理

    -  2xx 正常处理完成
        - 200 0k 请求成功

    -  3xx 重定向
        - 301 Moved Permanently 永久重定向
        - 302 Found 临时重写向

    -  4xx 客户端错误
        - 400 Bad Request 语法错误
        - 401 Unauthorized 权限未认证
        - 403 Forbidden 禁止访问
        - 404 Not Found 资源未找到

    -  5xx 服务器端错误
        - 500
